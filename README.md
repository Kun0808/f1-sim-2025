# 🏎️ F1 职业生涯模拟器

> 一款基于文字的F1赛车职业生涯模拟游戏，数据基于 **2025赛季** 真实车手、车队和赛历。

🎮 **在线游玩**: https://kun0808.github.io/f1-sim-2025/

## 📖 游戏介绍

你将扮演一位新晋F1车手，从选择背景故事、签约车队开始，经历一个又一个赛季的征战。通过训练提升能力、在比赛中做出策略决策、赢得积分和冠军，最终成为F1传奇。

## ✨ 核心功能

### 车手生涯
- **8种背景故事**: 青训新星、赞助车手、低级别霸主、跨界老将、临危受命、天才少年、王者归来、模拟器神童
- **10支车队自由选择**: 从迈凯伦到Kick Sauber，不同难度任你挑战
- **6项能力属性**: 速度、稳定、雨战、防守、进攻、赛道IQ

### 比赛系统
- **24站完整赛历**: 2025赛季全部24站大奖赛，从澳大利亚到阿布扎比
- **6站冲刺赛**: 中国、迈阿密、奥地利、比利时、美国、巴西、卡塔尔
- **4段比赛决策**: 轮胎策略、超车机会、队友博弈、最后冲刺
- **6种随机事件**: 安全车、对手退赛、引擎模式、黄旗区、工程师建议、DRS争夺
- **天气系统**: 晴天、多云、小雨、大雨，影响比赛表现

### 成长系统
- **训练中心**: 每站比赛前可训练一次，大幅提升指定属性
- **赞助商系统**: 6种赞助商提供不同属性加成和薪资加成
- **合同谈判**: 赛季结束后接收多支车队的报价，自由转会

### 成就系统
- 12个成就等你解锁：首胜、世界冠军、五冠王、雨神、逆转之王、铁人、忠诚等

### 其他
- **赛季结算**: 详细统计每赛季的排名、积分、领奖台、退赛等数据
- **自动存档**: 游戏进度自动保存在浏览器本地，随时可继续

## 🎮 玩法指南

1. **开始游戏**: 输入车手姓名，选择背景故事
2. **选择车队**: 根据难度标签选择适合自己的车队（强队容易赢，弱队有挑战）
3. **训练**: 每站比赛前可以训练一次，提升关键属性
4. **比赛周末**: 阅读新闻 → 排位赛 → 做出4个比赛决策 → 查看结果
5. **赛季结束**: 查看赛季总结，选择新车队合同
6. **重复**: 继续征战新赛季，追逐世界冠军
