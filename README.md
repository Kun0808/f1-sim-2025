# 🏎️ F1 职业生涯模拟器

> 一款基于文字的F1赛车职业生涯模拟游戏，数据基于 **2025赛季** 真实车手、车队和赛历。

🎮 **在线游玩**: https://kun0808.github.io/f1-sim-2025/

## 📖 游戏介绍

你将扮演一位新晋F1车手，从选择背景故事、签约车队开始，经历一个又一个赛季的征战。通过训练提升能力、在比赛中做出策略决策、应对赛道突发状况和车队内斗，管理赞助商和公关形象，赢得积分和冠军，最终成为F1传奇。

## ✨ 核心功能

### 车手生涯
- **两种开始方式**:
  - 🏁 **新建车手**: 创建自定义车手，选择背景故事和车队，并选择替代该车队的哪位现役车手
  - 👤 **扮演现役车手**: 从20位真实F1车手中任选一位，以他们的真实能力属性开启赛季
- **8种背景故事**: 青训新星、赞助车手、低级别霸主、跨界老将、临危受命、天才少年、王者归来、模拟器神童
- **10支车队自由选择**: 从迈凯伦到Kick Sauber，不同难度任你挑战
- **车手替代系统**: 新建车手时可选择替代车队的一号或二号车手，被替代的车手将离开围场
- **6项能力属性**: 速度、稳定、雨战、防守、进攻、赛道IQ（上限100，达到100显示MAX）

### 比赛系统
- **24站完整赛历**: 2025赛季全部24站大奖赛，从澳大利亚到阿布扎比
- **6站冲刺赛**: 中国、迈阿密、奥地利、比利时、美国、巴西、卡塔尔
- **4段比赛决策**: 轮胎策略、超车机会、队友博弈、最后冲刺
- **20种随机事件**（5大类）:
  - 🏁 **赛道突发**（6种）: 安全车、对手退赛、引擎模式、黄旗区、工程师建议、DRS争夺
  - 😤 **车队内斗**（4种）: 队友无视指令、策略分歧、车队威胁降级、队友暗讽
  - 📺 **媒体事件**（4种）: 媒体恶意剪辑、狗仔偷拍、名嘴批评、社交媒体风暴
  - 💥 **赛道戏剧**（5种）: 连环碰撞红旗、突降暴雨、技术故障、虚晃一枪、最后圈混乱
  - 🤫 **围场八卦**（3种）: 转会传闻、传奇点评、赞助商施压
- **天气系统**: 晴天、多云、小雨、大雨，影响比赛表现
- **高触发率**: 每场比赛平均触发1-2个戏剧性事件

### 成长系统
- **训练中心**: 每站比赛前可训练一次，单次提升4-7点属性（上限100，接近上限时有提示）
- **赞助商系统**:
  - 6种赞助商提供不同属性加成和薪资加成
  - 签约前弹出确认页面，展示年薪、年限、签字费等完整信息
  - 合同年限1-3年，签约后锁定不可更换
  - 提前解约需支付违约金
  - 签约奖金（0.5倍年薪）即时到账
- **合同谈判系统**:
  - 车队兴趣度（0-100%），基于玩家表现+声誉计算
  - 签字费、合同年限（1-3年）、总价值计算
  - 赛季目标与期望排名（争夺冠军/领奖台/积分区/完赛）
  - 合同条款（一号车手/平等对待/成绩奖金/续约选项/无条款）
  - 竞争对手系统（其他车手也在争席位，显示对方兴趣度）
  - 忠诚奖励（在同一车队效力越久，续约薪资越高）

### 车手社交系统
- **20位车手关系管理**: 与每位AI车手有独立的亲密度（0-100）
- **5种社交互动**: 一起晚餐($2M)、一起训练(免费)、模拟器对战($1M)、送礼物($5M)、公开赞扬(免费)
- **关系等级**: 敌对→冷淡→熟人→好友→挚友
- **转会影响**: 
  - 挚友(80+): 转会兴趣度+15，好友推荐你加入他们的车队
  - 好友(60+): 转会兴趣度+8
  - 敌对(<20): 转会兴趣度-10，对方会阻挠你加入
- **随机事件**: 好友车手可能在赛道上让位、敌对车手可能恶意阻挡、围场聚餐事件
- 队友初始亲密度更高（40-60），其他车手从20-50开始

### 车队领队系统
- **10位真实领队**: Andrea Stella(McLaren)、Christian Horner(Red Bull)、Toto Wolff(Mercedes)、Vasseur(Ferrari)等
- **领队事件**: 比赛中20%概率触发领队专属事件
  - 一对一谈话、施压要求成绩、信任投票、媒体战、暗示转会
  - 每个选择都会影响车队信任和领队关系
- **领队新闻**: 新闻系统中新增领队相关动态

### 经济系统
- **初始资金$15M**: 起步资金紧张，需要合理规划（原来$50M太容易）
- **薪水储值**: 玩家拥有金钱余额，每赛季开始自动获得车队薪水+赞助商收入
- **主界面显示余额**: 实时查看当前资金
- **公关管理系统**: 花费金钱提升声誉，6种公关活动可选:

  | 活动 | 花费 | 效果 |
  |------|------|------|
  | 🎤 新闻发布会 | $2M | 媒体关系 +8 |
  | 👋 车迷见面会 | $3M | 车迷人气 +10 |
  | 🍽️ 车队团建 | $3M | 车队信任 +10 |
  | 📱 社交媒体营销 | $4M | 车迷人气 +12, 媒体 +4 |
  | ❤️ 慈善活动 | $5M | 全属性小幅提升 |
  | ✨ 顶级公关团队 | $10M | 全属性大幅提升 |

### 声誉系统
- **4维声誉**: 车队信任、媒体关系、车迷人气、车手尊重
- 声誉影响合同报价、赞助商兴趣和车队关系
- 可通过公关活动花费金钱提升

### 成就系统
- 12个成就等你解锁：首胜、世界冠军、五冠王、雨神、逆转之王、铁人、忠诚等

### 车手详情查阅
- 积分榜中点击任意车手名字查看详细信息
- 显示：当前排名、积分、冠军/领奖台/最佳完赛/退赛次数
- 完整6项能力属性条形图
- 每位车手专属围场梗和基于属性的动态梗

### 趣味内容
- **28条围场趣闻**: 新闻系统中新增大量车手梗和趣味新闻
  - Verstappen打F1游戏不用自己、Hamilton的vegan食谱、Alonso的El Plan、Leclerc的钢琴演奏等
- **车手专属梗**: 每位车手在详情页有独特的趣味描述

### 其他
- **赛季结算**: 详细统计每赛季的排名、积分、领奖台、退赛等数据
- **一键模拟赛季**: 快速模拟剩余所有比赛，自动训练+自动决策，适合想快速推进的玩家
- **退役系统**: 随时可选择退役，展示完整生涯总结（生涯评级S+/S/A/B/C/D、胜率、领奖台率等）
- **自动存档**: 游戏进度自动保存在浏览器本地，随时可继续

## 🎮 玩法指南

1. **开始游戏**: 选择"新建车手"或"扮演现役车手"
2. **新建车手流程**: 输入姓名 → 选择背景故事 → 选择车队 → 选择替代哪位车手
3. **扮演现役车手**: 直接从20位真实车手中选择一位，以他们的能力开始
4. **训练**: 每站比赛前可以训练一次，提升关键属性（+4~7点，上限100，满级显示MAX）
5. **比赛周末**: 阅读新闻 → 排位赛 → 做出4个比赛决策（含随机事件） → 查看结果
6. **快速模拟**: 不想一场场跑？一键模拟剩余赛季，自动训练+最优决策
7. **赞助商管理**: 签约赞助商获得年薪加成和属性加成，合同期内锁定
8. **公关管理**: 花费金钱执行公关活动，提升车队信任/媒体关系/车迷人气/车手尊重
9. **车手社交**: 与围场车手互动提升亲密度，高亲密度影响转会和赛事事件
10. **查看对手**: 积分榜中点击车手名查看对手详细数据和围场梗
11. **赛季结束**: 查看赛季总结，从多支车队报价中选择新车队合同
12. **合同谈判**: 比较年薪、签字费、年限、赛季目标、合同条款和竞争对手
13. **退役**: 随时可以选择退役，查看生涯评级和完整数据总结
14. **重复**: 继续征战新赛季，追逐世界冠军
